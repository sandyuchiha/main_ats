How can I edit this code?

There are several ways of editing your application.

Option 1: Use the project UI

Visit the project dashboard and start prompting or editing directly.
Changes made via the project UI will be committed automatically.

Option 2: Use your preferred IDE

If you want to work locally using your own IDE, you can clone this repo and push changes back. Pushed changes will be reflected in the dashboard.

The only requirement is having Node.js & npm installed — recommended install via nvm.

Follow these steps:

# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and instant preview.
npm run dev

Option 3: Edit directly in GitHub

Navigate to the desired file(s).

Click the "Edit" button (pencil icon).

Make your changes and commit.

Option 4: Use GitHub Codespaces

Go to the main page of the repository.

Click on "Code".

Open the "Codespaces" tab.

Click "New codespace" to launch.

Edit, commit, and push changes.
